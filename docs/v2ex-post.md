开源了一个 AI 桌面浏览器 Drift Browser

最近用 Electron 33 写了个桌面浏览器，已开源。

主要功能：
- 内置 AI 对话（ChatGPT/Claude/Gemini 多模型）
- AI Agent 自动化操作
- 插件侧载系统（不用重新打包就能装插件）
- WebGL 星云背景 + 电影级开屏动画
- GitHub Releases 自动更新检测
- 广告拦截 + 标签冻结等性能优化

技术栈：Electron 33 + Chromium 130 + WebGL + Canvas
